  • Homura Discord Bot

    Homura is the first Discord Bot I developed. I coded this bot using Python, specifically the Discord.py extension. I started this project around 2020. With this bot, you can do simple moderation commands like banning, kicking, etc. or play games like shipping, dice, and much more!

  • TLX Discord Bot

    TLX Discord Bot is a bot written in Discord.py alongside Selenium for Web-Scraping. The purpose of this bot was to assist our school's Programming Club members to easily access profiles on the Programming Learning Platform. In this bot you call view the Rating, total AC problems and scores. TO spice things up, we also added a feature called badges!
